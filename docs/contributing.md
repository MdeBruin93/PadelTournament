# Contributing

Thank you for your interest in contributing to PadelTournament!

## Development Setup

### Requirements
- .NET 10 SDK
- Docker
- Git

### Running locally
```bash
dotnet run --project src/PadelTournament.AppHost
```

## Contribution Guidelines
- Keep the Core project free of infrastructure dependencies
- Add tests for new business rules
- Follow the existing project structure and conventions

## Pull Requests
- Keep changes small and focused
- Clearly explain the motivation and impact of the change
